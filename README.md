## 데이터베이스 학습 자료 개요

이 저장소는 `sakila` 샘플 데이터베이스를 기반으로 한 SQL 학습 자료와 프로젝트 메모를 정리한 공간입니다. 집합 함수, DDL/DML, 조인, 서브쿼리 등 핵심 개념을 실습 예제로 빠르게 확인할 수 있으며, `DailyMemo` 폴더에는 수업 기록과 아이디어 노트를 보관합니다.

## 폴더 및 파일 구성

- `AGG.sql` · `COUNT,MAX,MIN.sql`
  - `COUNT`, `SUM`, `AVG`, `MAX`, `MIN` 등 집계 함수 활용 예시를 담은 스크립트입니다.
- `GROUP_BY.sql` · `GROUP_BY_ex.sql`
  - 그룹화 및 `HAVING` 조건 실습 문제와 풀이가 포함돼 있습니다.
- `JOIN.sql`
  - 내부 조인과 외부 조인, self join 예제를 통해 다중 테이블 조회 패턴을 익힙니다.
- `ORDERBY_LIMIT.sql`, `WHERE.sql`, `CASE.sql`
  - 정렬, 필터링, 조건 분기 문법을 간단한 예제로 복습할 수 있습니다.
- `DDL.sql`, `DML.sql`, `VIEW.sql`, `INDEX.sql`
  - 스키마 생성/수정, 데이터 조작, 뷰 생성, 인덱스 관리 등 관리 작업 모음을 정리했습니다.
- `SubQuery.sql`, `Titanic_ex.sql`, `BDDB.sql`, `Untitled.sql`
  - 서브쿼리 연습, 탑승객 데이터를 활용한 분석 예제, 기타 실습 SQL을 모아 둔 파일입니다.
- `DailyMemo/`
  - 날짜별 SQL 수업 요약(예: `251028[sql_genimi].md`)과 `DATABASE_ERD.png`, 프로젝트 아이디어 기록(`안심먹거리DB실습.md`)을 보관합니다.
- `DB.txt`
  - 수업 중 참고한 메모나 요구 사항을 간단히 정리한 텍스트입니다.

## 활용 방법

1. `sakila` 데이터베이스가 준비돼 있다면, 필요한 `.sql` 파일을 순서대로 실행하며 결과를 확인해 보세요.
2. 새 개념을 공부할 때는 해당 파일을 열어 주석과 쿼리를 참고하고, 필요하면 자신만의 실습 노트를 추가하세요.
3. 프로젝트나 과제 아이디어는 `DailyMemo` 폴더에 날짜별로 기록하고, `DATABASE_ERD.png`를 활용해 구조를 시각적으로 점검하면 도움이 됩니다.

## 향후 확장 아이디어

- 실습 결과와 출력 예시를 README에 추가해 재현성을 높입니다.
- `DailyMemo`의 요구사항 정의를 기반으로 테이블 설계서와 실제 SQL 스크립트를 연동합니다.
- 반복적으로 사용하는 쿼리는 스니펫으로 정리하거나 Stored Procedure 예제로 확장해도 좋습니다.
